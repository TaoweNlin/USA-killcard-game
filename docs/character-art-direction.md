# 角色美术方向

本文记录运行时角色图的替换规范。角色图只负责人物底图，不包含 UI 卡框、身份章、血点、技能按钮或文字。

## 文件契约

- 路径：`public/assets/cards/characters/{characterId}.webp`
- 推荐尺寸：`750x1050`
- 比例：`5:7`
- 推荐头像裁切：从同一张角色立绘上半部裁切 `512x512`
- 格式：WebP 优先

## 画面要求

- 半身或胸像构图，人物占画面约 70%-85%。
- 脸部位于上半区，适配 `focalPoint: 50% 38%`。
- 保留边缘安全区，避免头顶、肩部、手部被 UI 裁掉。
- 背景可以有政治舞台、新闻灯光、国会剪影、红蓝光效等元素，但不能抢主体。

## 避免内容

- 不要在图片内绘制中文、英文、标语、logo、水印或可读文字。
- 不要绘制卡框、身份图标、血点、装备条或手牌 UI。
- 不要使用真实竞选 logo、党徽、商标或官方海报构图。
- 不要使用照片感过强的新闻图；当前游戏风格偏政治讽刺漫画卡图。

## 替换流程

1. 将新图导出为 `750x1050` WebP。
2. 覆盖对应 `public/assets/cards/characters/{characterId}.webp`。
3. 运行 `npm test` 和 `npm run build`。
4. 本地进入牌局，确认角色卡、开局选择界面和技能浮层没有遮挡主体。
